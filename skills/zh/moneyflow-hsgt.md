# 沪深港通资金

## 背景说明 — 中国跨境互联互通

**沪深港通（HSGT）**是外资交易 A 股、内地资金交易港股的主要渠道。2014 年开通沪港通、2016 年开通深港通。

### 两个方向
- **北向（north money）**：外资流入 A 股。**外资对中国权益资产态度的最重要指标。**
- **南向（south money）**：内地资金流入港股。

### 对全球投资者的重要性
- **北向是外资情绪的脉搏**：连续数周 `north_money` 为正说明外资在加仓。持续流出常常先于大盘走弱。
- **政策信号实时反映**：每日流量对政策变化（监管、税收、地缘）反应灵敏。注意突然反转。
- **没有更干净的替代**：不像美股有 13F 披露机构持仓，中国外资动向只能靠这个数据衡量。

### 关键字段解读
所有金额单位为**百万元**：
- **`north_money`**：北向资金合计净流入（沪股通 + 深股通）。正 = 净买入。
- **`south_money`**：南向资金合计净流入（内地投资港股）。
- **`hgt_ss`**：沪股通北向（外资 → 沪市）。
- **`hgt_sz`**：深股通北向（外资 → 深市）。
- **`ggt_ss`**：港股通南向（内地 → 港股，沪市渠道）。
- **`ggt_sz`**：港股通南向（内地 → 港股，深市渠道）。

### 注意事项
- 这是**汇总**数据；个股持仓在 `northbound_holdings` / `southbound_holdings`。
- 单日流量噪音大，看 5 日或 20 日均线更准确。
- 互联互通有每日额度上限，但近年很少触及。
- 中国法定假日期间互联互通会部分暂停，即使港股开市。

## SDK 方法

```python
client.moneyflow_hsgt(start_date=None, end_date=None, limit=100, offset=0)
```

## 返回类型

`pd.DataFrame` — 数值列为 `float64`。无数据时返回空 DataFrame。

## 示例

```python
df = client.moneyflow_hsgt(limit=3)
print(df[["trade_date", "north_money", "south_money"]])
```

## 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `start_date` | str | None | 起始日期，YYYY-MM-DD |
| `end_date` | str | None | 结束日期，YYYY-MM-DD |
| `limit` | int | 100 | 返回行数，最大 2000 |
| `offset` | int | 0 | 分页偏移量 |

## 返回字段

| 字段 | 说明 |
|------|------|
| `trade_date` | 交易日期 |
| `hgt_ss`, `hgt_sz` | 沪股通/深股通北向（百万元） |
| `ggt_ss`, `ggt_sz` | 港股通南向（百万元） |
| `north_money` | 北向资金净流入合计（百万元），正值=外资净买入 |
| `south_money` | 南向资金净流入合计（百万元） |

## 数据范围

- 起始日期：2014-11-17
- 每个交易日一条记录
- 外资对 A 股情绪的关键指标
- API 路径：`GET /v1/flows/moneyflow-hsgt`
