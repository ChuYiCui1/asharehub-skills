# 概念板块

## 背景说明 — A 股的主题驱动市场

**概念**是 A 股散户主导市场特有的主题分类。不同于传统行业分类（申万、GICS），概念反映**驱动 A 股板块轮动的交易主题和叙事**。

### 概念 vs 行业
| | 行业分类 | 概念 |
|---|---------|------|
| 来源 | 官方（申万/证监会） | 数据商整理（东方财富/同花顺） |
| 稳定性 | 长期稳定 | 根据市场题材新增/淘汰 |
| 示例 | 银行、医药、零售 | "AI 概念"、"金砖概念"、"华为 Mate 概念"、"ChatGPT 概念" |
| 个股归属 | 一只股票一个行业 | 一只股票可属多个概念 |

### A 股中概念的重要性
- **板块轮动驱动**：A 股上涨经常伴随"概念轮动"——资金以周为单位在不同热门题材之间流动。
- **新闻驱动**：一个新政策或产品发布就能催生新概念，几天内成分股拉涨 20-50%。
- **散户关注**：散户交易概念，不交易基本面。追踪概念热度可揭示散户资金流向。

### 常见概念类别
- **科技叙事**：AI、ChatGPT、机器人、半导体、自动驾驶
- **政策主题**：一带一路、金砖、共同富裕、国企改革
- **供应链**：特定公司产业链（如"苹果链"、"特斯拉链"）
- **宏观事件**：飓风、疫情、俄乌战争等

### 关键字段解读
- **`ts_code`**：概念指数代码（东方财富格式如 `BK0425.DC`）。
- **`name`**：概念中文名（如"ChatGPT概念"）。
- **`leading`**：当日领涨成分股。
- **`pct_change`**：概念指数日涨跌幅。
- **`up_num` / `down_num`**：上涨/下跌成分股数量。
- **`level`**：概念层级。

### 注意事项
- 概念归属**由数据商定义、带主观性**。同一题材在不同数据商之间成分股可能不同。
- 概念**不一定基本面纯粹**——一只"机器人概念"股可能机器人业务只占营收 5%。
- 热门概念在叙事高潮后倾向**均值回归**。

## SDK 方法

```python
client.concepts(ts_code=None, start_date=None, end_date=None, limit=100, offset=0)
```

## 返回类型

`pd.DataFrame` — 数值列为 `float64`。无数据时返回空 DataFrame。

## 示例

```python
df = client.concepts(limit=3)
print(df[["trade_date", "name", "pct_change", "leading"]])
```

## 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `ts_code` | str | None | 板块代码，如 `BK0425.DC` |
| `start_date` | str | None | 起始日期，YYYY-MM-DD |
| `end_date` | str | None | 结束日期，YYYY-MM-DD |
| `limit` | int | 100 | 返回行数，最大 5000 |
| `offset` | int | 0 | 分页偏移量 |

## 返回字段

| 字段 | 说明 |
|------|------|
| `ts_code` | 板块代码（如 BK0425.DC） |
| `trade_date` | 交易日期 |
| `name` | 板块名称 |
| `leading` | 领涨股 |
| `leading_code` | 领涨股代码 |
| `pct_change` | 涨跌幅 % |
| `leading_pct` | 领涨股涨幅 |
| `total_mv` | 总市值（万元） |
| `turnover_rate` | 换手率 % |
| `up_num` | 上涨家数 |
| `down_num` | 下跌家数 |
| `idx_type` | 指数类型 |
| `level` | 级别 |
