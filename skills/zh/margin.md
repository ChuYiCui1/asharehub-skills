# 融资融券

## 背景说明 — 与美股 margin 的差异

**A 股融资融券存在但运作方式与美股差异很大。**

### 与美股 margin 的关键差异
| | 美股 Margin | A 股融资融券 |
|---|-------------|------------|
| 可交易股票 | 大多数股票 | **白名单制**（约 2000 只） |
| 融券（做空） | 容易，按 locate 借券 | **受限**——只能从券商池借券，常常无券可借 |
| 保证金比例 | ~50%（Reg T） | 由券商设定，通常 50-60% |
| 散户使用 | 常见 | **投机者重度使用**，是杠杆情绪指标 |

### `rzye` 和 `rqye` 是什么
- **`rzye`（融资余额）**：未偿还的融资买入贷款总额。**`rzye` 上升 = 多头杠杆累积**，最受关注的指标。
- **`rqye`（融券余额）**：未偿还的融券（做空）总额。通常远小于 `rzye`，因为做空困难。

### A 股投资者为什么关注
- **情绪代理指标**：全市场 `rzye` 总额是最干净的情绪指标之一，急升常与市场顶部重合。
- **个股信号**：`rzye` 快速上升的股票被杠杆投机者激进买入——动量强但反转风险高。
- **强制平仓风险**：股价下跌时融资盘可能被强制平仓，加速下跌。

### 关键字段解读（金额均为元）
- **`rzye`（融资余额）**：未偿还融资买入余额（多头杠杆未平仓量）。
- **`rqye`（融券余额）**：融券余额。
- **`rzmre`（融资买入额）**：当日融资买入额。
- **`rqyl`（融券余量）**：融券余量股数。
- **`rzche`（融资偿还额）**：当日融资偿还额。
- **`rqchl` / `rqmcl`**：融券偿还/卖出量。
- **`rzrqye`（融资融券余额）**：融资融券合计余额（元）。

### 注意事项
- 只有列入"融资融券标的"的股票才能进行两融交易。使用前先确认股票是否在标的池。
- 融券券源常常受限，`rqye` 可能因无券可借而偏低。
- 融资余额是**收盘后数据**，非日内实时。

## SDK 方法

```python
client.margin(ts_code=None, start_date=None, end_date=None, limit=100, offset=0)
```

## 返回类型

`pd.DataFrame` — 数值列为 `float64`。无数据时返回空 DataFrame。

## 示例

```python
df = client.margin(ts_code="000001.SZ", limit=3)
print(df[["trade_date", "rzye", "rqye", "rzrqye"]])
```

## 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `ts_code` | str | None | 股票代码，如 `000001.SZ`（平安银行）、`600519.SH`（贵州茅台） |
| `start_date` | str | None | 起始日期，YYYY-MM-DD |
| `end_date` | str | None | 结束日期，YYYY-MM-DD |
| `limit` | int | 100 | 返回行数，最大 5000 |
| `offset` | int | 0 | 分页偏移量 |

## 返回字段

| 字段 | 说明 |
|------|------|
| `trade_date` | 交易日期 |
| `ts_code` | 股票代码 |
| `name` | 股票名称 |
| `rzye` | 融资余额 |
| `rqye` | 融券余额 |
| `rzmre` | 融资买入额 |
| `rqyl` | 融券卖出量 |
| `rzche` | 融资偿还额 |
| `rqchl` | 融券偿还量 |
| `rqmcl` | 融券卖出量 |
| `rzrqye` | 融资融券余额 |

## 数据范围

- 起始日期：2020-01-02
- API 路径：`GET /v1/market/margin`
